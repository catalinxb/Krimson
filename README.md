# Krimson 📉
> A sleek, high-performance trading journal for modern traders.

Krimson helps you track your trades, analyze your performance, and refine your edge with a clean, intuitive interface. Built with React and Vite for maximum speed.

## ✨ Features
*   **Trade Logging:** Quickly record entries, exits, and position sizing.
*   **Performance Analytics:** View your win rate, profit factor, and equity curve.
*   **Strategy Tagging:** Categorize trades by setup to see what's actually working.
*   **Fast UI:** Powered by Vite for a near-instant developer and user experience.
